Fontify overview

Fontify allows to see Google Web Fonts in action - either on this page or your own! Simply hover over the toolbar at the top of the window to change fonts.

When you're happy with the results, click the "Use fonts" link to haz teh codez.

If you want to share your work with someone else, use the "Share" link to get a custom URL that will preload your selected fonts.

Fontify on this site

This is an active demo. Using the toolbar above should make real-time changes to this site's fonts. The "h3" selector should hit all of the headers on this side of the page, while the "section p" selector gets all of the paragraphs (except the code blocks) on this side.

Go ahead. Give it a whirl.

Fontify on your site

As a jQuery plugin, Fontify requires ... jQuery. So, be sure to do something like this:

<script type="text/javascript" src="jquery.js"></script>
<script type="text/javascript" strc="jquery.fontify.js"></script>

Then you'll be able to use Fontify to preview fonts on your own site with some code like this:

<script type="text/javascript">
    $(document).ready(function() {
        $('.headlines, p').fontify({ apiKey: 'YOUR-GOOGLE-API-KEY' });
    });
</script>

For more about the apiKey option, see Options below.

More about Fontify

Fontify is a jQuery plugin. It was developed against jQuery 1.8.3 and tested exclusively in Chromium 22.0.1229.94 on Ubuntu 12.10, so if you want to tell me about your troubles (which is welcome), be sure to include a bunch of information like that.

Options

  • apiKey - Required. This is a Google API key from the Google API console. The value can be found on the "API Access" tab, and "Web Fonts Developer API" must be enabled on the "Services" tab.
    • Values: See Google API console
  • position - This docks the Fontify toolbar to either the top or bottom of the window.
    • Values: 'top', 'bottom'
    • Default: 'top'
  • activeOpacity - The opacity of the Fontify toolbar on mouse hover.
    • Values: 0.0 to 1.0
    • Default: 0.9
  • inactiveOpacity - The opacity of the Fontify toolbar on mouse leave.
    • Values: 0.0 to 1.0
    • Default: 0.2
